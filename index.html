Welcome to LanPlan (beta)!

Welcome to LanPlan, a platform which will help you plan and find nearby LAN parties. With this project we hope to encourage more socialising (in real-life) in the gaming community. At the moment, we are busy with other (boring) work, but hopefully we will be able to continue developing this project.

The idea of LanPlan originated from an under 18s hackathon which ran throughout the UK for a week called Young Rewired State, an awesome annual event which I urge anyone who is interested to check out.

Well, thanks for dropping by. As I've said, the project is still in early beta, so don't expect everything to run smoothly - we've still got a lot to refine!

Bye for now, Chris and Chris
